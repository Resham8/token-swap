# Solana Token Swap

A decentralized application that enables instant token swaps on the Solana blockchain.

## Tech Stack
- React/Next.js
- Solana Web3.js
- Solana wallet adapter
- TypeScript

## Features
- Instant token swaps
- Wallet integration (Phantom)
- Real-time price updates
- Low transaction fees on Solana

## Setup
```bash
npm install
npm run dev
```
